MM_HELP_0001: How to get mobile beacon 3D orientation via the modem

 

 

To get 3D orientation from the paired beacons via the modem you should enable 'IMU via modem' option as shown below:

 

This mobile beacon should be the left beacon of the pair and have a higher address than the right beacon.

For example if your paired beacons have addresses n10 and n11, the n11 should be the left one and the 'IMU via modem' option should be enabled for n11.
